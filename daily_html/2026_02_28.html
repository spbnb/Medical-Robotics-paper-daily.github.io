Utilizing LLMs for Industrial Process Automation

Abstract (English): A growing number of publications address the best practices to use Large Language Models (LLMs) for software engineering in recent years. However, most of this work focuses on widely-used general purpose programming languages like Python due to their widespread usage training data. The utility of LLMs for software within the industrial process automation domain, with highly-specialized languages that are typically only used in proprietary contexts, remains underexplored. This research aims to utilize and integrate LLMs in the industrial development process, solving real-life programming tasks (e.g., generating a movement routine for a robotic arm) and accelerating the development cycles of manufacturing systems.

摘要（中文）: 近年来，越来越多的出版物探讨了使用大型语言模型（LLMs）进行软件工程的最佳实践。 然而，由于其训练数据使用的广泛性，大部分工作集中于像 Python 这样被广泛使用的通用编程语言。在工业过程自动化领域，大型语言模型在软件方面的效用，特别是对于通常只在专有环境中使用的、高度专业化的语言，仍然有待探索。 本研究旨在利用大型语言模型并将其集成到工业开发过程中，解决实际编程任务（例如，生成机器人手臂的运动程序），并加速制造系统的开发周期。

TLDR: This paper explores the use of LLMs in the industrial process automation domain, focusing on specialized programming languages and real-world automation tasks like robotic arm control.

MediX-R1: Open Ended Medical Reinforcement Learning

Abstract (English): We introduce MediX-R1, an open-ended Reinforcement Learning (RL) framework for medical multimodal large language models (MLLMs) that enables clinically grounded, free-form answers beyond multiple-choice formats. MediX-R1 fine-tunes a baseline vision-language backbone with Group Based RL and a composite reward tailored for medical reasoning: an LLM-based accuracy reward that judges semantic correctness with a strict YES/NO decision, a medical embedding-based semantic reward to capture paraphrases and terminology variants, and lightweight format and modality rewards that enforce interpretable reasoning and modality recognition. This multi-signal design provides stable, informative feedback for open-ended outputs where traditional verifiable or MCQ-only rewards fall short. To measure progress, we propose a unified evaluation framework for both text-only and image+text tasks that uses a Reference-based LLM-as-judge in place of brittle string-overlap metrics, capturing semantic correctness, reasoning, and contextual alignment. Despite using only $\sim51$K instruction examples, MediX-R1 achieves excellent results across standard medical LLM (text-only) and VLM (image + text) benchmarks, outperforming strong open-source baselines and delivering particularly large gains on open-ended clinical tasks. Our results demonstrate that open-ended RL with comprehensive reward signals and LLM-based evaluation is a practical path toward reliable medical reasoning in multimodal models. Our trained models, curated datasets and source code are available at https://medix.cvmbzuai.com

摘要（中文）: 我们介绍了MediX-R1，一个开放式的强化学习（RL）框架，用于医学多模态大型语言模型（MLLM），它能够提供临床层面扎实、自由形式的答案，超越了多项选择题的形式。 MediX-R1 使用基于组的强化学习和专为医学推理定制的复合奖励，对基线的视觉-语言主干网络进行微调：一个基于LLM的准确度奖励，通过严格的是/否决策来判断语义的正确性；一个基于医学嵌入的语义奖励，用于捕捉释义和术语变体；以及轻量级的格式和模态奖励，用于加强可解释的推理和模态识别。这种多信号设计为开放式输出提供了稳定且信息丰富的反馈，而传统的、可验证的或仅基于多项选择题的奖励则相形见绌。为了衡量进展，我们提出了一种统一的评估框架，用于纯文本任务和图像+文本任务，该框架使用基于参考的LLM作为评判者来替代脆弱的字符串重叠度量，从而捕捉语义正确性、推理和上下文一致性。尽管只使用了约 5.1 万个指令示例，MediX-R1 在标准医学LLM（纯文本）和VLM（图像+文本）基准测试中均取得了优异的成绩，超越了强大的开源基线，尤其是在开放式临床任务中取得了显著的提升。我们的结果表明，带有全面奖励信号和基于 LLM 评估的开放式强化学习是多模态模型中实现可靠医学推理的有效途径。我们训练的模型，整理的数据集和源代码可在 https://medix.cvmbzuai.com 获取。

TLDR: MediX-R1 is an open-ended RL framework for medical MLLMs that utilizes group-based RL and a composite reward system for improved medical reasoning and benchmark performance, especially on open-ended tasks.

Interface-Aware Trajectory Reconstruction of Limited Demonstrations for Robot Learning

Abstract (English): Assistive robots offer agency to humans with severe motor impairments. Often, these users control high-DoF robots through low-dimensional interfaces, such as using a 1-D sip-and-puff interface to operate a 6-DoF robotic arm. This mismatch results in having access to only a subset of control dimensions at a given time, imposing unintended and artificial constraints on robot motion. As a result, interface-limited demonstrations embed suboptimal motions that reflect interface restrictions rather than user intent. To address this, we present a trajectory reconstruction algorithm that reasons about task, environment, and interface constraints to lift demonstrations into the robot's full control space. We evaluate our approach using real-world demonstrations of ADL-inspired tasks performed via a 2-D joystick and 1-D sip-and-puff control interface, teleoperating two distinct 7-DoF robotic arms. Analyses of the reconstructed demonstrations and derived control policies show that lifted trajectories are faster and more efficient than their interface-constrained counterparts while respecting user preferences.

摘要（中文）: 辅助机器人为重度运动障碍人士提供了行动能力。通常，这些用户通过低维界面控制高自由度机器人，例如使用一维吹吸界面操作六自由度机械臂。这种不匹配导致在给定时间只能访问控制维度的一个子集，从而对机器人运动施加了非预期的和人为的约束。因此，受界面限制的示教数据嵌入了次优运动，这些运动反映了界面限制而不是用户的意图。为了解决这个问题，我们提出了一种轨迹重建算法，该算法能推理任务、环境和界面约束，从而将示教数据提升到机器人的完整控制空间中。我们使用通过二维操纵杆和一维吹吸控制界面进行远程操作两种不同的七自由度机械臂所执行的、受日常生活活动启发的真实示教数据来评估我们的方法。对重建的示教数据和导出的控制策略的分析表明，提升后的轨迹比其受界面约束的对应轨迹更快、更高效，同时尊重用户偏好。

TLDR: The paper introduces a trajectory reconstruction algorithm to improve robot learning from demonstrations using low-dimensional interfaces, addressing the limitations imposed by interface constraints and generating faster, more efficient robot trajectories.

简要说明: 该论文介绍了一种轨迹重建算法，该算法通过低维界面改进从演示中进行的机器人学习，解决了界面约束带来的限制，并生成更快、更高效的机器人轨迹。

SeeThrough3D: Occlusion Aware 3D Control in Text-to-Image Generation

Abstract (English): We identify occlusion reasoning as a fundamental yet overlooked aspect for 3D layout-conditioned generation. It is essential for synthesizing partially occluded objects with depth-consistent geometry and scale. While existing methods can generate realistic scenes that follow input layouts, they often fail to model precise inter-object occlusions. We propose SeeThrough3D, a model for 3D layout conditioned generation that explicitly models occlusions. We introduce an occlusion-aware 3D scene representation (OSCR), where objects are depicted as translucent 3D boxes placed within a virtual environment and rendered from desired camera viewpoint. The transparency encodes hidden object regions, enabling the model to reason about occlusions, while the rendered viewpoint provides explicit camera control during generation. We condition a pretrained flow based text-to-image image generation model by introducing a set of visual tokens derived from our rendered 3D representation. Furthermore, we apply masked self-attention to accurately bind each object bounding box to its corresponding textual description, enabling accurate generation of multiple objects without object attribute mixing. To train the model, we construct a synthetic dataset with diverse multi-object scenes with strong inter-object occlusions. SeeThrough3D generalizes effectively to unseen object categories and enables precise 3D layout control with realistic occlusions and consistent camera control.

摘要（中文）: 我们将遮挡推理识别为3D布局条件生成中一个基础但被忽视的方面。它对于合成具有深度一致的几何形状和大小的、部分遮挡的对象至关重要。虽然现有方法可以生成符合输入布局的逼真场景，但它们通常无法精确建模对象间的遮挡。我们提出了SeeThrough3D，一个用于3D布局条件生成且显式建模遮挡的模型。我们引入了一种遮挡感知3D场景表示(OSCR)，其中对象被描绘为放置在虚拟环境中的半透明3D盒子，并从所需的相机视角渲染。透明度编码隐藏的对象区域，使模型能够进行遮挡推理，而渲染的视点在生成过程中提供显式的相机控制。通过引入一组从我们渲染的3D表示中导出的视觉令牌，我们对预训练的基于流的文本到图像生成模型进行条件化。此外，我们应用掩码自注意力来精确地将每个对象边界框绑定到其对应的文本描述，从而实现多个对象的准确生成，而不会发生对象属性混合。为了训练模型，我们构建了一个合成数据集，其中包含具有强烈对象间遮挡的各种多对象场景。SeeThrough3D能够有效地推广到未见过的对象类别，并实现精确的3D布局控制，同时保证逼真的遮挡效果和一致的相机控制。

TLDR: SeeThrough3D introduces an occlusion-aware 3D scene representation for text-to-image generation, enabling better object occlusion and camera control in generated images based on text prompts and 3D layouts.

简要说明: SeeThrough3D 提出了一种用于文本到图像生成的、具有遮挡感知能力的 3D 场景表示，可以在基于文本提示和 3D 布局生成的图像中实现更好的对象遮挡和相机控制。

Evaluating Zero-Shot and One-Shot Adaptation of Small Language Models in Leader-Follower Interaction

Abstract (English): Leader-follower interaction is an important paradigm in human-robot interaction (HRI). Yet, assigning roles in real time remains challenging for resource-constrained mobile and assistive robots. While large language models (LLMs) have shown promise for natural communication, their size and latency limit on-device deployment. Small language models (SLMs) offer a potential alternative, but their effectiveness for role classification in HRI has not been systematically evaluated. In this paper, we present a benchmark of SLMs for leader-follower communication, introducing a novel dataset derived from a published database and augmented with synthetic samples to capture interaction-specific dynamics. We investigate two adaptation strategies: prompt engineering and fine-tuning, studied under zero-shot and one-shot interaction modes, compared with an untrained baseline. Experiments with Qwen2.5-0.5B reveal that zero-shot fine-tuning achieves robust classification performance (86.66% accuracy) while maintaining low latency (22.2 ms per sample), significantly outperforming baseline and prompt-engineered approaches. However, results also indicate a performance degradation in one-shot modes, where increased context length challenges the model's architectural capacity. These findings demonstrate that fine-tuned SLMs provide an effective solution for direct role assignment, while highlighting critical trade-offs between dialogue complexity and classification reliability on the edge.

摘要（中文）: 领导者-追随者交互是人机交互（HRI）中一个重要的范式。然而，对于资源受限的移动和辅助机器人而言，实时分配角色仍然具有挑战性。虽然大型语言模型（LLMs）在自然通信方面表现出潜力，但其尺寸和延迟限制了设备端部署。小型语言模型（SLMs）提供了一种潜在的替代方案，但其在HRI中角色分类方面的有效性尚未得到系统评估。本文提出了一个针对领导者-追随者通信的SLM基准测试，引入了一个新颖的数据集，该数据集源自已发布的数据库，并使用合成样本进行增强以捕获特定于交互的动态。我们研究了两种适应策略：提示工程和微调，并在零样本和单样本交互模式下进行了研究，并与未经训练的基线进行了比较。使用Qwen2.5-0.5B进行的实验表明，零样本微调实现了稳健的分类性能（准确率86.66%），同时保持了低延迟（每个样本22.2毫秒），显著优于基线和提示工程方法。然而，结果也表明在单样本模式下性能下降，其中增加的上下文长度挑战了模型的架构能力。这些发现表明，经过微调的SLM为直接角色分配提供了一种有效的解决方案，同时突出了边缘设备上对话复杂性和分类可靠性之间的关键权衡。

TLDR: This paper benchmarks the effectiveness of small language models (SLMs) for leader-follower role classification in human-robot interaction, finding that zero-shot fine-tuning of Qwen2.5-0.5B achieves high accuracy with low latency, but performance degrades in one-shot scenarios.

Generalized Rapid Action Value Estimation in Memory-Constrained Environments

Abstract (English): Generalized Rapid Action Value Estimation (GRAVE) has been shown to be a strong variant within the Monte-Carlo Tree Search (MCTS) family of algorithms for General Game Playing (GGP). However, its reliance on storing additional win/visit statistics at each node makes its use impractical in memory-constrained environments, thereby limiting its applicability in practice. In this paper, we introduce the GRAVE2, GRAVER and GRAVER2 algorithms, which extend GRAVE through two-level search, node recycling, and a combination of both techniques, respectively. We show that these enhancements enable a drastic reduction in the number of stored nodes while matching the playing strength of GRAVE.

摘要（中文）: 摘要： 广义快速动作价值估计（GRAVE）已被证明是蒙特卡洛树搜索（MCTS）算法族中用于通用游戏博弈（GGP）的一种强大变体。然而，其对在每个节点存储额外胜/访统计数据的依赖使得它在内存受限的环境中应用不切实际，从而限制了其实际应用。在本文中，我们分别引入了GRAVE2、GRAVER和GRAVER2算法，它们通过两级搜索、节点回收以及两种技术的结合来扩展GRAVE。我们表明，这些增强能够在匹配GRAVE博弈强度的同时，大幅减少存储的节点数量。

TLDR: This paper introduces GRAVE2, GRAVER, and GRAVER2 algorithms that improve upon the GRAVE algorithm for General Game Playing (GGP) by significantly reducing memory usage without sacrificing playing strength.

简要说明: 本文介绍了GRAVE2，GRAVER和GRAVER2算法，它们改进了用于通用游戏玩法（GGP）的GRAVE算法，通过显着减少内存使用量而不牺牲游戏强度。
